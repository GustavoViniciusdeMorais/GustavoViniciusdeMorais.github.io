API Product Description

GET /products/{product_id}

Retrieve information about a specific product.

Request

{ "product_id": 123 }

Response

{ "id": 123, "name": "Sample Product", "price": 19.99 }

POST /products

Create a new product.

Request

{ "name": "New Product", "price": 29.99 }

Response

{ "id": 456, "name": "New Product", "price": 29.99 }
